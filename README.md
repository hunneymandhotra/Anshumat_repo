# RecruitAI: AI-Powered Recruitment Experience

A modern recruitment platform designed to eliminate the limitations of traditional PDF-based hiring. RecruitAI uses a conversational, AI-driven interface to capture structured candidate data, ensuring a more efficient and unbiased evaluation process.

## 🚀 Vision
Traditional resumes (PDFs) are static, difficult to parse, and often lead to biased decisions. RecruitAI replaces the "Upload Resume" button with a **Smart Profile Builder** that understands candidate potential through guided interaction.

### Why Resume-based Hiring is Broken:
- **Poor Parsing**: Inconsistent data structure makes automated screening unreliable.
- **Formatting Bias**: Candidates with better "Resume Design" skills are favored over those with better technical skills.
- **Static Content**: Resumes don't reflect real-time growth or granular project impact.

### How AI Solves This:
- **Structured Data Capture**: AI takes raw, conversational input ("I led a team at Microsoft...") and structures it into standardized schema (Role, Company, Impact Metrics).
- **Skill Evolution**: Instead of a list of words, AI maps experiences to a unified skill taxonomy.
- **Biased-Free Screening**: Recruiters compare candidates on verified impact data, not just formatting.

## 📦 Getting Started

### 1. Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### 2. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
Accessible at: `http://localhost:5173`

### 3. Setup Backend
```bash
cd backend
npm install
npm run dev
```
Accessible at: `http://localhost:5001`
